# Security Policy

## Reporting A Vulnerability

Email `contact@arvantacyber.com` with the subject `Code Hunter security report`.

Please include:

- Affected product area.
- Reproduction steps.
- Impact.
- Any relevant version or environment detail.
- Whether the report includes sensitive information.

Do not open a public GitHub issue for sensitive vulnerability details.

## Public Repository Boundary

This repository is public product documentation. Do not submit:

- Product source code.
- License keys or activation material.
- API keys, tokens, cookies, or session data.
- Customer source code or reports.
- Private product screenshots.
